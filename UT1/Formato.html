Formato de textos

Estas son unas de las etiquetas más comunes, básicas y más utilizadas para dar formato en HTML:

<b> y <strong>: Negrita

<b> aplica negrita sin indicar ninguna importancia semántica.


<strong> aplica negrita e indica que el texto es importante.



<i> y <em>: Cursiva


<i> aplica cursiva sin indicar ninguna importancia semántica.


<em> aplica curiva e indica que el texto es importante.



<u>: Subrayado


Se utiliza para subrayar el texto.


<strike> o <s>: Texto tachado


Tacha el texto, indicando que esta borrado, o que no es relevante.



<sup>: Superíndice


Se utiliza para elevar el texto, como por ejemplo los exponentes matemáticos.



<sub>: Subíndice


Se utiliza para poner texto en la posición de subíndice, esta se utiliza en fórmulas químicas.
