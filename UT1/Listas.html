Listas en HTML

Listas no ordenadas (<ul>):
  1. Se utilizan para listas de elementos sin un orden específico.
  2. Los elementos se muestran con viñetas u otros símbolos antes de cada ítem.
  3. Cada elemento de la lista se define con la etiqueta <li> (list item)


Listas ordenadas (<ol>):
  • Se utilizan para listas en que el orden de los elementos son relevantes.
  • Los elementos se ennumeran automáticamente (1,2,3,etc.).
  • Cada elemento de la lista se define con la etiqueta <li> (list item)

Atributos de listas
  1. Atributo type (para listas ordenadas <ol>):
    • Se puede utilizar para definir el tipo de ennumeración de la lista:
      • 1: Numeros (predeterminado)
      • A: Mayúsculas.
      • a: Minúsculas.
      • I: Números romanos en mayúsculas.
      • i: Números romanos en minúsculas.
  2. Atributo start (para listas ordenadas <ol>):
    • Define el número o letra por el cual queremos comenzar la ennumeración.
