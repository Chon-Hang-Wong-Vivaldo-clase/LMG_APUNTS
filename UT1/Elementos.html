Elementos HTML y anidamientos

Etiqueta (Tag)
  • Es la parte que define el inicio y final de un componente en HTML.
  • Se delimita con los símbolos < >


Elemento
  • Un elemento incluye la etiqueta de apertura, dentro de este el contenido (si hay), y por último la etiqueta de cierre.
  • Los elementos pueden estar vacios, o pueden contener otro elementos dentro.

Elementos anidados
  • Con anidamientos se refiere a colocar un elemento HTML dentro de otro elemento.
  • Es importante seguir correctamente el orden de las etiquetas: la primera que se abre es la última que se debe cerrar
