Inserción de imagenes

  • La etiqueta <img> se utiliza para insertar imagenes en una página web.
  • No tiene etiqueta de cierre, ya que es un elemento auto-cerrado.
  • Es escencial proporcionar información sobre la imágen, para que se muestre de manera correcta y mejorar su accesibilidad.

Atributos de imagenes:
  1. src (source):
    • Define la fuente o el camino de la imágen.
    • Puede ser un camino relativo o una URL completa.
  2. alt (alternative text):
    • Proporciona un texto alternativo que se muestra si la imágen no se carga o si es leída por lectores de pantallas.
    • Es fundamental para la accesibilidad de una y para el SEO.
  3. width y height:
    • Especifica el ancho (width) y la altura (height) de la imágen en píxeles o porcentajes.
    • Es importante para controlar la medida de la imagen en la página

Buenas prácticas
  • Siempre utiliza el atributo alt para garantizar una buena accesibilidad y SEO.
  • Evita insertar imagenes muy grandes sin reducir su medida, ya que eso puede afectar al rendimiento de la página.
  • Si las imagenes son responsive, utiliza porcentaje en vez de píxeles por width y height.
