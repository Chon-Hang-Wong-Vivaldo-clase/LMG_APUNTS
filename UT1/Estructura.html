Estructura básica de un documento HTML

Estructura general
  1. Un documento HTML se basa en una estructura jerárquica de etiquetas que definen las diferentes partes de una página.
  2. Cada página HTML sigue una estructura similar, empezando por el <!DOCTYPE>, seguido de las etiquetas <HTML>, <head>, <body>.
