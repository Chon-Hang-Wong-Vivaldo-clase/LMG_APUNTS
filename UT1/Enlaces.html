Inserción de imagenes

  • La etiqueta <a> se utiliza para crear enlaces (links) a otras páginas web, documentos, secciones dentro de la misma página o cualquier otro recurso.
  • El elemento escencial para los enlaces es el atributo href que define la URL a la cual redirige al usuario.
  • El texto entre la etiqueta de apertura y la de cierre es lo que los usuarios verán y podrán hacer click para ser rederigidos.

Atributos de los enlaces:
  1. href (hypertext reference):
    • Define la URL de destino del enlace
    • Puede ser un camino absoluto (una URL completa) o relativo (camino dentro del mismo dominio).
  2. target:
    • Define donde se abrirá el enlace: en la misma pestaña, una nueva pestanya o una nueva ventana.
    • Los valores más habituales son:
      • _blank: Abre el enlace en una pestaña nueva.
      • _self: Anre el enlace en la misma pestaña (valor predeterminada).

Buenas prácticas
  • Utilizar enlaces descriptivos en lugar de genéricos como "Click aquí", esto mejora la accesibilidad y el SEO.
    • Correcto: "Visita la página de Contacto".
    • Incorrecto: "Click aquí".
  • Utiliza target="_blank" únicamente cuando sea realmente necesario, ya que abrir enlaces en nuevas pestañas puede confundir a algunos usuarios.
  • Siempre asegurarse que los enlaces estén accesibles añadiendo una descipción adequada para los usuarios con lectores de pantallas.
