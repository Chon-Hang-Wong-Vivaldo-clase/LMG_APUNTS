Introducción al HTML

¿Qué es HTML?
  1. HTML (HyperText Markup Language) es el lenguaje de marcas para crear páginas web.
  2. Define la estructura de la página con etiquetas que el navegador interpreta para mostrar el contenido
    3. contenido.
Breve Historia
  1. Creado por Tim Berners-Lee el 1991.
  2. HTML 2.0 (1995): Primer estándar oficial.
  3. HTML5 (2014): Última versión con nuevas funcionalidades como apoyo para video, audio y gráficos.
