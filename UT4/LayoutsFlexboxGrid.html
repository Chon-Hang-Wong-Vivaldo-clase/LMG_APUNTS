CSS Avanzado - 4. Layouts avanzados con Flexbox y Grid:

  1. Flexbox: casos comunes:

    1. Cuando quieres alinear elementos en fila o columna, como botones en una barra de navegación o imágenes en una galería.
    2. Cuando quieres distribuir el espacio entre elementos en una fila (por ejemplo, centrando un elemento o espaciándolos uniformemente).
    3. Cuando los elementos deben crecer o reducirse para ocupar el espacio disponible, como en una barra lateral o una fila de tarjetas.

  2. Grid: casos comunes:

    1. Cuando quieres crear una página con una estructura compleja (como una página con cabecera, barra lateral, contenido principal y pie de página).
    2. Cuando tienes un diseño en formato de cuadrícula con filas y columnas, como un tablero de fotos o un diseño de tarjetas de productos.
    3. Cuando quieres controlar exactamente dónde va cada elemento en la cuadrícula (puedes colocar elementos en cualquier celda de la cuadrícula).

  1. Flexbox avanzado: alineación, "flex-shrink", "align-self": Flexbox es una herramienta muy flexible para crear diseños en fila o columna, donde los elementos se alinean y se ajustan automáticamente según el espacio disponible.

    1. flex-shrink: Controla cómo se reduce el tamaño de un elemento cuando no hay suficiente espacio disponible. Valores más altos permiten que el elemento se reduzca más.

    2. align-self: Permite que un solo elemento dentro de un contenedor flex tenga una alineación diferente al resto de los elementos. Los valores pueden ser: flex-start, flex-end, center, baseline o stretch.


  2. Grid layout. Creación de cuadrículas: CSS Grid Layout es una herramienta más potente que Flexbox cuando se trata de crear layouts en dos dimensiones (filas y columnas). Grid permite colocar elementos en una cuadrícula con mucha precisión, controlando el tamaño de las filas y columnas, y permitiendo que los elementos se ajusten al espacio disponible.

    Propiedades claves en CSS Grid:

    • grid-template-column y grid-template-rows: Definen el número y el tamaño de las columnas y filas dentro del contenedor grid.
    • grid-column: Permiten que un elemento ocupe varias columnas o filas dentro de la cuadrícula.
    • gap: Define el espacio entre filas y columnas de la cuadrícula.

  3. Uso combinado de Flexbox: Aunque Flexbox y Grid se pueden utilizar por separado, a menudo es útil combinarlos para aprovechar lo mejor de ambos. Por ejemplo, puedes usar Grid para definir la disposición general de una página (filas y columnas) y utilizar Flexbox dentro de algunas áreas para ajustar la alineación de los elementos.

    1. El layout general se estructura con Grid, diviendo la página en columnas.
    2. Dentro de una de las celdas del Grid, se utiliza Flexbox para alinear los elementos en una fila.
