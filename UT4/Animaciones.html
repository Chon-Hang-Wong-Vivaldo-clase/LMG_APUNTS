CSS Avanzado - 7. Animaciones y transiciones:

Las animaciones y transiciones en CSS son herramientas poderosas que permiten añadir movimiento e interacción visual a las páginas web, mejorando la experiencia del usuario. Con estas técnicas, puedes controlar cómo los elementos cambian sus propiedades (como color, posición u opacidad) de forma suave y fluida.

  1. Uso de transciciones (transition): Las transiciones CSS permiten que los cambios en las propiedades de un elemento, como el color, la opacidad, el tamaño o la posición, sean suaves en lugar de bruscos. Cuando una propiedad cambia, puedes definir la velocidad del cambio utilizando transition.

    Sintaxis básica de transition:
    1. Propiedad: La propiedad CSS que quieres animar (por ejemplo, background-color, transform, opacity, etc.).
    2. Duración: Cuánto tiempo durará la transición (por ejemplo, 0.5s, 2s).
    3. Función de temporización: Controla cómo será la velocidad de la transición (por ejemplo, ease, linear, ease-in-out).
    4. Retardo: Opcional, define cuánto tiempo tardará en comenzar la transición (por ejemplo, 1s).

  2. Propiedad de transición (transition-property): Si quieres animar varias propiedades simultáneamente, puedes especificarlas individualmente con transition-property o usar all para aplicar la transición a todas las propiedades que cambien.

  3. Animaciones clave con @keyframes: Las animaciones en CSS permiten crear movimientos más complejos que las transiciones. Utilizan la regla @keyframes para definir puntos intermedios del movimiento o cambio de propiedad, y se aplican mediante la propiedad animation.

    Sintaxis de @keyframes y animación:
    • Nombre de la animación: El nombre de la animación definida con @keyframes.
    • Duración: El tiempo que durará la animación.
    • Función de temporización: Controla la velocidad de la animación (por ejemplo, linear, ease, etc.).
    • Retardo: Opcional, cuánto tiempo tardará en comenzar la animación.
    • Ciclos: Cuántas veces se repetirá la animación (infinite para repetirla indefinidamente).
    • Dirección: Opcional, determina si la animación se ejecuta de forma normal o inversa (normal, reverse, alternate).
  4. Propiedades de animación (animation-duration, animation-delay): Las propiedades de animación permiten personalizar como se comporta una animación. Las más importantes son:
    • animation-duration: Controla cuanto tiempo dura la animación
    • animation-delay: Define un retraso antes que la animación comience
    • animation-iteration-cout: Controla cuantas veces se repetirá la alineación (1, 3, infinite)
    • animation-direction: Define si la animación se arranca hacia delante (normal), hacia atrás (reverse) o alterna (alternate)
