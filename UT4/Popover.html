CSS Avanzado - 5. Popover API:

El Popover API es una funcionalidad reciente que permite a los desarrolladores crear contenido emergente (popovers) de forma nativa, utilizando solo HTML y CSS, sin necesidad de JavaScript. Esta API facilita la creación de componentes como menús, sugerencias o notificaciones que aparecen sobre el contenido existente.

  1. Atrubutos HTML relevantes:

      popover: Atributo global que designa un elemento como popover
      1. "auto" (por defecto): El popover se muestra y se oculta automáticamente
      2. "manual": El control de la visibilidad del popover se gestiona manualmente

      popovertarget: Atributo que especifica el ID del elemento popover que se controlará

      popovertargetaction: Define la acción a realizar sobre el popover.
      1. "show": muestra el popover
      2. "hide": oculta el popover
      3. "toggle": alterna entre mostrar y ocultar el popover
