CSS Avanzado - 2. Selectores avanzados:

Los selectores avanzados en CSS permiten aplicar estilos con mayor precisión y flexibilidad, seleccionando elementos específicos en función de su posición, relación con otros elementos o atributos.

  • Selectores de atributo: Estos permiten aplicar estilos a elementos HTML que contienen un atributo o valor de atributo determinado. Por ejemplo:

  • Selectores combinados: Permiten seleccionar elementos que tienen una relación específica con otros elementos dentro del documento. Los combinadores más comunes son:

      1. Combinador de descendientes ( ): selecciona todos los elementos que sean descendientes (hijos, nitetos...) de un elemento padre.
      2. Combinador de hijos directos (>): selecciona únicamente los hijos directos de un elemento padre
      3. Combinador de hermanos adjecientes (+): selecciona el elemento que está justo después de otro elemento hermano
      4. Combinador de hermanos generales (-): selecciona todos los hermanos que siguen un elemento determinado dentro del mismo padre

    2. Selectores pseudo-clase: Las pseudo-clases son selectores que se utilicen para aplicar estilos a un elemento en un estado o condición particular. Esto permite cambiar el estilo de un elemento cuando el usuario interactúa o cuando el elemento cumple ciertos criterios.

      1. :hover: aplica estilos cuando el usuario pasa con el ratón por encima del elemento
      2. :nth-child(): selecciona elementos en función de su posiciónm entre sus hermanos
      3. :first-of-type: selecciona el primer elemento de su tipo dentro de su padre
      4. :not(): selecciona todos los elementos excepto aquellos que coincidan con el selector específico

    3. Selectores pseudo-elemnt: Los pseudo-elementos permiten aplicar estilos a una parte específica de un elemento, como su contenido antes o después, o una línea de texto. Los pseudo- elementos más comunes son ::before y ::after.

      1. ::before: añade contenido o estilos antes del contenido real de un elemento
      2. ::after: añade contenido o estilos después del contenido real de un elemento
