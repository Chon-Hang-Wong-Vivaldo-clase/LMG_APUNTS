CSS básico - 5. Márgenes, rellenos y bordes en CSS:

Los márgenes (margin), los rellenos (padding) y los bordes (border) son parte del modelo de caja en CSS, que describe como se disponen los elementos en la página. Estas propiedades permiten ajustar el espacio exterior e interior de un elemento, así como definir los límites visuales con un borde.

  1. Márgenes (margin): La propiedad margin controla el espacio exterior de un elemento, es decir, el espacio que hay entre un elemento y los otros elementos que lo rodean. Los márgenes se pueden definir de manera individual para cada uno de los cuatro lados del elemento (superior, derecho, inferior, izquierdo) o de manera global para todos los lados a la vez.

    Ejemplo: 
                        
p {
    margin: 10px;
}

  2. Rellenos: (padding) La propiedad padding controla el espacio interior de un elemento, es decir, el espacio entre el contenido de un elemento (como texto o imágenes) y sus bordes. Igual que con los márgenes, los rellenos se pueden definir para todos los lados o para cada uno individualmente.

    Ejemplo: 
                        
p {
    padding: 10px;
}

  3. Bordes (border):La propiedad border se utiliza para añadir un borde alrededor deun elemento. Esta propiedad tiene varios subatributos:

    • Anchura del borde (border-width): define el grosor del borde.
    • Estilo del borde (border-style): puede ser solid (sólido), dashed (rayado), dotted (puntitos), etc.
    • Color del borde (border-color): especifica el color del borde.



      • Ejemplo: 
                      
p {
    border: 10px solid black;
}
