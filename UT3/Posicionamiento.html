CSS básico - 8. Posicionamiento básico en CSS:

El CSS ofrece diversas técnicas para controlar la ubicación de los elementos en una página web. La propiedad clave para el posicionamiento es "position", que tiene diversos valores para definir como se posicionará un elemento respecto a su contenedor.

  1. La propiedad position:

    La propiedad position permite especificar el método de posicionamiento de un elemento. Los valores más comunes son:

    1. static (por defecto): los elementos se posicionan de manera normal, siguiendo el flujo del documento. No se puede ajustar manualmente con coordenadas.
    2. relative: se posiciona el elemento respecto a su posición original en el flujo del documento. Se pueden utilizar coordenadas para moverlo ligeramente (arriba, abajo, izquierda o derecha)
    3. absolute: el elemento se sale del flujo del documento y se posiciona respeto a su elemento contenedor más cercano que tenga una posición definida. Si no hay ningún contenedor definido, se posiciona respecto a la página
    4. fixed: el elemento se sale del flujo del documento y se posiciona respecto a la ventana del navegador. Se mantiene fijo incluso cuando el usuario realiza desplazamientos.
    5. sticky: el elemento se comporta como relativo hasta que llegue a un punto en el desplazamiento de la página, después del fixed

  2. Propiedades de posicionamiento (top, right, bottom, left):

    Cuando se define la propiedad como relative, absolute, fixed o sticky, puedes utilizar las propiedades top, right, bottom, y left para especificar la distancia del elemento respecto a sus límites.

  3. Posicionamiento absoluto (poisiotn: absolute): con (position: absolute) el elemento se saca completamente del flujo normal del documento y se posiciona respecto a su elemento contenedor más cercano con posición: relative, absolute o fixed.

  4. Posicionamiento fija (position: fixed): los elementos con (position: fixed) se mantienen en una ubicación fija a la pantalla incluso cuando el usuario realiza desplazamiento. Se posicionan respecto a la ventana del navegador.
  5. Posicionamiento pegajoso (position: sticky): los elementos con (position: sticky) se comportan como relative hasta que el usuario se desplaza por la página y llega a un punto específico, en el que el elemento se comporta como fijo.
  6. Propiedad z-index: la propiedad z-index se utiliza para controlar el orden de superposición de los elementos posicionados (relative, absolute, fixed o sticky). Los valores más altos de z-index colocan el elemento delante de otros elementos con valores inferiores.
