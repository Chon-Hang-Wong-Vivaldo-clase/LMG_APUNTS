CSS básico - 2. Aplicación de estilos:

El CSS se puede aplicar a un documento HTML de tres maneras principales: en línea, incrustado (dentro del HTML) y externo (mediante un archivo CSS separado). Cada una de estas formas tiene sus propósitos y usos.

  1. Estilos en linea (inline styles): Los estilos en línea se aplican directamente a un elemento HTML utilizando el atributo style dentro de la misma etiqueta del elemento. Este método es adecuado para aplicar estilos a un solo elemento de manera rápida, pero no se recomienda para páginas grandes, ya que dificulta el mantenimiento.

    Ejemplo: 
                        
<p style="color: red; font-size: 20px;">Este párrafo con estilos en línea</p>
    Resultado:

    Este párrafo con estilos en línea

    • Ventajas:
      • Aplicación rápida y directa al elemento

    • Desventajas:
      • Dificulta el mantenimiento de las webs grandes porque se debería aplicar el estilo a cada elemento.
      • No aprovecha la capacidad de reutilizar estilos.

  2. Estilos incrustados (Embedded styles): Los estilos incrustados se definen dentro de la etiqueta <style>, que se coloca en la sección <head> de un documento HTML. Estos estilos son útiles cuando se quieren aplicar reglas CSS que afecten a todos los elementos de una página concreta, sin necesidad de un archivo CSS externo.

    3. Ejemplo: 
                        
<!DOCTYPE html>
<html lang="es">
    <head>
        <meta charset="UTF-8">
        <title>Estilos incrustados</title>
        <style>
            p {
                color: blue;
                font-size: 18px;
            }
            h1 {
                text-align: center;
            }
        </style>
    </head>
    <body>
        <h1>Este es un titulo centrado.</h1>
        <p>Este párrafo tendrá texto de color azul y tamaño de 18 píxeles.</p>
    </body>
</html>
    Resultado: Estils incrustats

    Este es un titulo centrado

    Este párrafo tendrá texto de color azul y tamaño de 18 píxeles.

    • Ventajas:
      • Más eficiente que los estilos en línea, ya que se pueden aplicar estilos a diversos elementos de una misma página desde un solo lugar.

    • Desventajas:
      • Solo afecta a la página donde se aplica, no se reutiliza entre varias páginas.
      • Si la página crece demasiado, puede dificultar la lectura y el mantenimiento.

  3. Estilos externos (External stylesheets): Los estilos externos se almacenan en un archivo CSS separado y se enlazan al documento HTML utilizando la etiqueta <link> dentro de la sección <head>. Este es el método más recomendado para la mayoría de sitios web, especialmente si constan de múltiples páginas.

    4. Ejemplo: HTML 
                        
<!DOCTYPE html>
<html lang="ca">
    <head>
        <meta charset="UTF-8">
        <title>Estils externs</title>
        <link rel="stylesheet" href="style.css">
    </head>
    <body>
        <h1>Bienvenidos a mi página web</h1>
        <p>Este parágrafo tendrá un tamaño de 16 píxeles</p>
    </body>
</html>
    Ejemplo: CSS 
                        
                        p {
                            color: green;
                            font-size: 16px;
                            }
                            h1 {
                            text-align: center;
                            color: darkblue;
                            }
    Resultado:

    Bienvenidos a mi página web

    Este parágrafo tendrá un tamaño de 16 píxeles
