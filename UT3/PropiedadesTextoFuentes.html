CSS básico - 4. Propiedades de texto y fuentes:

El CSS permite controlar como se muestran los textos dentro de una página web. Con diversas propiedades, pueden cambiar la fuente, el color, la alineación, la medida de los textos y otros aspectos visuales con tal de mejorar la legibilidad y la estética del contenido.

  1. Propiedad font-family: La propiedad font-family define el tipo de letra (tipografía) que se utilizará para un elemento de texto. Puedes especificar una o más fuentes, y el navegador aplicará la primera que este disponible en el sistema del usuario.

    Ejemplo: 
                        
p {
    font-family: "Arial", sans-serif;
}

  2. Propiedad font-size: La propiedad font-size controla el tamaño de la letra. Los valores se pueden modificar en diversas unidades Ej. pixeles (px), ems (em), porcentajes (%)...

    Ejemplo: 
                        
h1 {
    font-size: 32px;
}
p {
    font-size: 14px;
}

  3. Propiedad color: La propiedad "color" define el color del texto. Se pueden utilizar nombres de colores predefinidos, valores hexadecimales, o sistemas de color como el RGB o RGBA.

    Ejemplo: 
                        
h1 {
    color: red;
}
p {
    color: blue;
}

  4. Propiedad text-align: La propiedad text-align se utiliza para controlar la alineación del texto dentro de un elemento. Los valores más comunes son: left (izquierda), right (derecha), center (centrado) y justify (justificado).

    Ejemplo: 
                        
h1 {
    text-align: center;
}
p {
    text-align: justify;
}

  5. Propiedad font-weight: La propiedad font-weight controla el grosor de la letra. Los valores comunes son:
    • nomral: por defecto
    • bold: texto en negrita
    • Números entre 100 y 900, donde 4oo es el valor por defecto y 700 representa negrita.


    Ejemplo: 
                        
p {
    font-weight: bold;
}

  6. Propiedad font-style: La propiedad font-style controla el estilo de la fuente. Estos son los valores más comunes:
    • normal: text nomral
    • bold: texto en negrita
    • oblique: texto inclinado


    Ejemplo: 
                        
span {
    font-style: italic;
}
