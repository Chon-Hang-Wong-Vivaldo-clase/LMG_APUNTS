CSS básico - 3. Sintaxis básica:

  1. Regla CSS: es una instrucción que define como se ha de aplicar un estilo a uno o más elementos HTML dentro de una página web.

  2. Selector: indica el elemento HTML al cual se aplicaron los estilos.

  3. Propiedades: define la característica que quieras modificar (como el color, el tamaño del texto, el margen, etc.)

  4. Valor: asigna un valor específico a la propiedad (como ahora, "azul" para el color, "16px" para el tamaño del texto, etc.)

  5. Declaración CSS: es una instrucción dentro de una regla de CSS que define una característica específica de estilo para un elemento HTML.

Selectores en CSS

Los selectores son una parte fundamental del CSS, porque define que elementos HTML seran afectados por las reglas de estilo. Hay diversos tipos de selectores, pero los más comunes en los aprendizajes básicos son:

  1. Selectores de tipos (Type selector): Este selector aplica estilos a todos los elemetnos HTML del mismo tipo. Este código aplica el color verde a todos los títulos <h1> de la página.

    Ejemplo: 
              
h1 {
  color: green;
}

  2. Selectores de clase (Class selector): Los selectores de clase permiten aplicar estilos a elemento HTML con una clase específica. Se utiliza un punto (.) para seleccionar una clase. Cualquier elemento HMTL que tenga el atributo class="mi-clase" tendrá un fondo amarillo. Por ejemplo:

    Ejemplo: 
              
.mi-clase {
  background-color: yellow;
}

  3. Selector de ID (ID selector): El selector de ID se utiliza para aplicar estilos a un único elemento HTML que tenga un identificador específico. Se utiliza el símbolo "#". Es importante recordar que un ID ha de ser único en la página.

    Ejemplo: 
              
#mi-id {
  text-align: center;
}

  4. Selector universal: El selector universal (*) aplica estilos a todos los elementos de una página. Este ejemplo elimina los margenes y rellenos de todos los elementos de la página, útil para establecer un estilo uniforme.

    Ejemplo: 
              
* {
  margin: 0;
  padding: 0;
}

Conceptos importantes

  • Herencias: Los hijos tienen los estilos de sus padres, así no hace falta escribir los estilos por todo.

    Ejemplo: 
              
body {
  color: blue;
}

h1 {
  color: red;
}
  • Especificidad: Cuando hay conflictos de estilos, el navegador aplica únicamente el más específico.

    Ejemplo: 
              
.parrafo {
  color: blue;
}

.article .parrafo{
  color: red;
}
  • Cascada: Todo estilo sobreescribe el anterior. La "C" de CSS significa Cascada.

El CSS permite controlar como se muestran los textos dentro de una página web. Con diversas propiedades, pueden cambiar la fuente, el color, la alineación, la medida de los textos y otros aspectos visuales con tal de mejorar la legibilidad y la estética del contenido.

  1. Propiedad font-family: La propiedad font-family define el tipo de letra (tipografía) que se utilizará para un elemento de texto. Puedes especificar una o más fuentes, y el navegador aplicará la primera que este disponible en el sistema del usuario.
  2. Propiedad font-size: La propiedad font-size controla el tamaño de la letra. Los valores se pueden modificar en diversas unidades Ej. pixeles (px), ems (em), porcentajes (%)...
