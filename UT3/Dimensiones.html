CSS básico - 7. Dimensiones y cajas en CSS:

El CSS utiliza el modelo de caja para representar los elementos de una página web. Cada elemento se considera como una caja rectangular que contiene el contenido, rellenados, márgenes y bordes. Es importante entender este modelo porque controla como se distribuyen y se ajustan los elementos dentro de la página.

El modelo de las cajas se componen de cuatro partes principales:

  1. Contenido: el contenido real del elemento, como texto, imagenes u otros elementos.

  2. Relleno (padding): el espacio entre el contenido y el borde.

  3. Borde (border): el límite al rededor del relleno.

  4. Margen (margin): El espacio externo entre el borde y otros elementos.

Estos son algunos de los aspectos más importantes de las cajas:

  1. Propiedades de dimensiones (width y height): Las propiedades width (acho) y height (altura) se utilizan para definir el tamaño de un elemento. Estos valores se pueden definir en unidades fijas como píxeles (px) o unidades relativas como el porcentaje (%), que dependen del tamaño de su elemento contenedor.

  2. El modelo de caja en CSS: el modelo de caja se establece como se calculan las dimensiones de un elemento teniendo en cuenta las propiedades de margenes, rellenos, y bordes. Por defecto, el tamaño de width y height únicamente afecta el contenido. Para incluir el relleno y el borde dentro de las dimensiones del elemento, se puede utilizar la propiedad box-sizing.

    3. Propiedad box-sizing

    La propiedad box-sizing controla como se calculan las dimensiones totales de elementos. Hay dos opciones principales:

    1. content-box (valor por defecto): Únicamente incluye el contenido en las dimensiones mientras que los bordes y los rellenos se añaden a parte.
    2. border-box: inluyen el contenido, el relleno y el borde dentro de las dimensiones específicas.

  3. Control del desbordamiento (overflow): cuando el contenido de un elemento es más grande que las dimensiones especificadas, el CSS ofrece la propiedad overflow para controlar como se muestren el contenido que desborda.

    Los valores más comunes son:
    • visible: el contenido que desborde se muestra fuera del elemento (valor por defecto)
    • hidden: el contenido que desborde se recorta y no se muestra
    • scroll: muestra barras de desplazamiento sin que se desborde el contenido
    • auto: muestra barras de desplazamiento únicamente si el contenido se desborda
