CSS básico - 9. Diseño con Flexbox:

Flexbox es un sistema de diseño en CSS que proporciona una manera muy eficiente de distribuir el espacio entre los elementos y alinearlos dentro de un contenedor, incluso cuando la medida de los elementos es desconocida o dinámica. Flexbox es ideal para crear diseños de páginas adaptativas, puesto que facilita la alineación, distribución y ordenación de los elementos dentro de un contenedor.

  1. El contenedor flex (display: flex): El punto de partida para utilizar Flexbox es establecer (display: flex) en el elemento contenedor. Este contenedor se conoce como contenedor flex, y sus elementos hijos se conocen como elementos flex. Una vez activado el (display: flex), puedes utilizar diversas propiedades para ajustar como se distribuyen los elementos dentro del contenedor


  2. Dirección del flujo (flex-direction): por defecto, los elementos flex se posicionan en fila horizontal. Puedes cambiar la dirección con la propiedad flex-direction.

    • row: los elementos se colocan en fila (horizontal)
    • row-reverse: se colocan en fila pero en orden inverso.
    • column: los elementos se colocan en columna (vertical)
    • column-reverse: los elementos se colocan en columna pero en orden inverso.

  3. Alineación horizontal (justify-content): la propiedad justify-content, controla la alineación de los elementos a lo largo del eje principal (horizontal por defecto). Puedes distribuir los elementos flex de diferentes maneras dentro del contenedor.

    • flex-start: los elementos se alinean al comienzo del contenedor
    • flex-end: se alinean al final del contenedor
    • space-between: los elementos se distribuyen de forma uniforme con espacio entre ellos
    • space-around: los elementos tienen espacios alrededor de cada uno

  4. Alineación vertical (align-items): la propiedad align-items controla la alineación de los elementos a lo largo del eje transversal (vertical, si la dirección es horizontal). Esta propiedad determina como se alinean los elementos flex dentro del contenedor en relación con su altura.

    • stretch: los elementos se estiran para rellenar el contenedor
    • flex-start: los elementos se alinean en la parte superior del contenedor
    • flex-end: se alinean en la parte inferior del contenedor
    • center: los elementos se centran verticalmente dentro del contenedor
    • baseline: los elementos se alinean según su línea de base de texto

  5. Control del espacio que ocupa cada elemento (flex-grow): La propiedad flex-grow especifica como los elementos flex crecen dentro del contenedor para ocupar el espacio disponible. Cada elemento se puede asignar un valor numérico que indica cuánto tiene que crecer en relación con los otros elementos. Si todos los elementos tienen flex-grow: 1, el espacio disponible se distribuirá de manera uniforme.
