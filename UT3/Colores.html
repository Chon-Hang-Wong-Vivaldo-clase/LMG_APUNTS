CSS básico - 6. Colores y fuentes en CSS:

El CSS proporciona diversas propiedades para aplicar colores y fuentes a los elementos HTML. Estas propiedades son escenciales para mejorar el aspecto visual y la legibilidad de la página web.

  1. Propiedad color: La propiedad color se utiliza para cambiar el color del texto dentro de un elemento. Puedes definir los colores de diversas maneras:

    1. Nombre de colores: utilizar nombres predefinidos como red, blue, green, etc.

    2. Colores hexadecimales: utilizar un código hexadecimal que comienza con "#" seguido de seis carácteres.

    3. RGB: utilizar valores de rgb() donde cada componente (r=Rojo, g=Verde, b=Azul) y se especifica en valores entre el 0 y el 255.

    4. RGBA: igual que rgb(), pero con un cuarto de valor para la opacidad (valores entre 0.0 y 1).

  2. Propiedad background-color: La propiedad background-color se utiliza para establecer el color de fondos de un elemento. Igual que la propiedad color, se puede definir colores utilizando nombres, valores hexadecimales, RGB o RGBA.

  3. Propiedad background: La propiedad background es una manera abreviada de especificar diversas propiedades de fondos, como el color, la imágen de fondos, repetición, posición, etc.

  4. Propiedad background-image: Esta propiedad permite añadir una imágen como fondo de un elemento. Se puede especificar la ubicación de la imágen con una URL.

Si quieres que la imágen se repita, o se muestre de otra manera, se puede utilizar otras propiedades complementarias:

  1. background-repeat: Controla si la imágen de fondo se repite. Los valores pueden ser repeat, no-repeat, repeat-x (horizontal) o repeat-y (vertical).

  2. background-position: Define la posición de la imágen de fondo (por ejemplo; center, top, left...)

  3. background-size: Permite ajustar el tamaño de la imágen (por ejemplo, cover o contain)
