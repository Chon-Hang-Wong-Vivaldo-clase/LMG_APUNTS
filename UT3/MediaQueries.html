CSS básico - 10. Introducción Media Queries:

Media Queries son una característica poderosa del CSS que permite adaptar el diseño de una página web en función de las características del dispositivo que lo está visualizando, como la medida de la pantalla, la resolución o la orientación. Esta técnica es clave para implementar diseños responsivos que ofrecen una experiencia óptima tanto en dispositivos móviles como en tabletas u ordenadores de sobremesa.

  1. ¿Qué son las Media Queries? Una media query es una condición que se añade al CSS para aplicar reglas de estilo solo cuando se cumplan ciertas características del dispositivo o de la ventana del navegador. El ejemplo más común es aplicar estilos diferentes en función de la anchura de la pantalla.

  2. Aplicación básica de Media Queries para pantallas de diferentes tamaños: Un uso típico de las media queries es adaptar el diseño a diferentes anchuras de pantalla. Por ejemplo, puedes ajustar los estilos para pantallas grandes y pequeñas.

  3. Tipos de condiciones más comunes en las Media Queries:

    • Ancho de la pantalla (min-width y max-width): estas condiciones se utilizan para aplicar estilos cuando la ventana del navegador tiene una anchura mínima (min-width) o máxima (max-width).
    • Altura de la pantalla (min-height y max-width): estas condicions funcionan de manera similar al de los ancho, pero se basan en la altura de la página
    • Orientación de la pantalla: puedes utilizar la condición orientation para aplicar estilos diferentes si la pantalla está en orientación horizontal (landscape) o vertical (portrait).

  4. Uso de múltiples condiciones en una Media Query: Puedes combinar varias condiciones en una sola media query mediante las palabras clave and o only para especificar estilos muy concretos.
