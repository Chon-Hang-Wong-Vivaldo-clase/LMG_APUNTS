Multimedia con HTML

HTML5 introdujo maneras nuevas y eficientes de integrar multimedia a las páginas web, incluyendo imagenes, vídeos y audio sin necesidad de plug-ins externos con Flash.
Esto ha mejorado tanto la compatibilidad como el uso en multiples dispositivos, especialmente móviles.

Imagenes en HTML:

La etiqueta <img> se utiliza para insertar imagenes en una página web.

Esta etiqueta es auto-cerrada (no tiene etiqueta de cierre)

Atributos principales:

  1. src: Especifica la ruta de la imágen.
  2. alt: Proporciona un texto alternativo para la imágen, útil para la accesibilidad y el SEO.
  3. width y height: Controlan la medida de la imágen.

Vídeos en HTML:

La etiqueta <video> se utiliza para insertar videos en una página web, de manera nativa sin plug-ins externos.

Esta etiqueta es auto-cerrada (no tiene etiqueta de cierre)

Atributos principales:

  1. src: Especifica la ruta del vídeo.
  2. controls: Muestra los controles del vídeo (play, pause, volume, etc.).
  3. autoplay: Hace que el vídeo se reproduzca automáticamente cuándo se carga la página.
  4. loop: Hace que el vídeo se reproduzca en bucle.
  5. muted: Reproduce el vídeo sin sonido

Audios en HTML5:

La etiqueta <audio> se utiliza para insertar audios de manera nativa.

Atributos principales:

  1. src: Especifica la ruta del audio.
  2. controls: Muestra los controles del audio (play, pause, volume, etc.).
  3. autoplay: Hace que el audio se reproduzca automáticamente cuándo se carga la página.
  4. loop: Hace que el audio se reproduzca en bucle.
  5. muted: Reproduce el aduio sin sonido inicialmente
Buenas prácticas
  1. Proporciona formatos de ficheros comparibles con los diferentes navegadores para audio y vídeo (MP4, WebM, etc.).
  2. Utiliza el atributo alt para las imagenes para mejorar la accecibilidad y optimizar el SEO.
  3. No utilices el atributo autoplay en audio y vídeo, ya que puede interrumpir la experiencia del usuario.
  4. Controla la medida de los ficheros multimedia para evitar tiempos de carga larga y mejorar la experiencia del usuario.
